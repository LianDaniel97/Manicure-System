# frontend-capstone-team-21
**Nail Studio Booking System**

A dynamic appointment management and booking platform for nail studios.

**Project Description**

This system is an interactive platform that allows studio clients to book appointments simply, quickly, and intuitively. The system is based on a Multi-step Wizard process, guiding the user from service selection to final confirmation by cross-referencing real-time data regarding staff availability and offered services.


**Technologies Used**

The project was developed using modern client-side web technologies:
1. HTML5: Structured semantic forms and wizard stages.
2. CSS: Modern styling using Flexbox and Grid, CSS Variables for consistent branding, and smooth transitions between stages.
3. JavaScript (Vanilla): Business logic management, DOM manipulation, data validation, and State Management.
4. Git & GitHub: Version control with a history of over 20 commits documenting the development process.

**Key Features**

1. Multi-step Wizard: Seamless booking process divided into stages to prevent visual overload and improve user experience (UX).
2. Dynamic Staff Filtering: The system displays only staff members qualified for the specific service selected (based on specialty logic).
3. Real-time Availability: Displays only available time slots and prevents double-booking.
4. Advanced Data Validation: Ensures valid phone number formats (10 digits) and prevents illegal characters in text fields.
5. Responsive Design: Fully compatible with mobile devices, tablets, and desktops.
6. Summary & Review Screen: Displays a full summary of appointment details and pricing for user verification before final submission.

**How to Run**

1. Download the Project: Clone the repository or download the project folder as a ZIP file.
2. Launch: Navigate to the main folder and open index.html in any modern web browser (Chrome, Edge, Safari).
3. Testing the Flow:
   
    Step 1: Enter your name, phone number, and select a service.
  
    Step 2: Select a staff member and a date (past dates are automatically disabled).
  
    Step 3: Review the appointment summary and click "Confirm Booking."
  
4. Confirmation: Upon completion, a success message will be displayed.
